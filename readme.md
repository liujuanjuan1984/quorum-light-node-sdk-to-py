# quorum-light-node-sdk-to-py

## Install

(require **Python** and **Nodejs**)

```
cd quorum-light-node-sdk-to-py
pip3 install -r requirements.txt                                                
npm install
```

## Usage

### add group
```
python3 addGroup.py
```


### get group
```
python3 getGroup.py
```

### list groups
```
python3 listGroups.py
```

### create trx
```
python3 createTrx.py
```

### get trx
```
python3 getTrx.py
```

### list contents
```
python3 listContents.py
```
